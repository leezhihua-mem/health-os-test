# 🥔 土豆智能体 - 健康预警测试页面

**在线测试：** https://leezhihua-mem.github.io/health-os-test/

---

## 功能说明

**25个预警模型：**
- ✅ 血压预警（BPV+PP+MH三维评分）
- ✅ 血糖预警（空腹+餐后+HbA1c）
- ✅ 心率预警（静息心率+心率恢复+心率变异性）
- ✅ 血脂预警（LDL-C+HDL-C+TG）
- ✅ 体重预警（BMI+腰围+体脂率）
- 📝 其他预警模型待实现

## 使用说明

1. 选择预警类型
2. 输入健康数据
3. 点击"计算预警"
4. 查看预警结果
